# Chillify - Web App Music Player (Frontend)

Chillify is a **web-based music player application**, developed as the frontend part of the **Chillify - Web App Music Player** project.  
This frontend focuses on user experience, music player state management, and the **practical application of classic Design Patterns** using **React + TypeScript**.

The main goals of the frontend are:

- Provide a smooth and intuitive music listening experience
- Manage player state and behavior in a clean, maintainable way
- Apply well-known design patterns in real-world frontend development

---

## Tech Stack

- **Frontend**: React.ts (TypeScript)
- **Styling**: Tailwind CSS

---

## Project Structure

```bash
frontend/
├── public/             # Static files and root HTML
├── src/
│   ├── assets/         # Images and static assets
│   ├── components/     # Reusable UI components
│   ├── layouts/        # Layout components
│   ├── pages/          # Main pages (Home, My Playlists, ...)
│   ├── App.tsx         # Root component
│   └── main.tsx        # Application entry point
├── package.json        # Project metadata and dependencies
└── README.md           # Project documentation
```

## Getting Started

1. **Install dependencies:**

   ```bash
   cd frontend
   npm install
   ```

2. **Run the application:**

   ```bash
   npm run dev
   ```
